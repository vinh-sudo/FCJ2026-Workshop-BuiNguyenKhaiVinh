# “Cloud Mastery 2026 #2 DevOps”

### Event Objectives

- Provide an overview of Kubernetes, IaC, and Elixir.
- Equip participants with practical knowledge for building, deploying, and operating cloud-based systems.
- Bridge theory and practice through live demonstrations.

### Speakers & Topics

**1. Bao Huynh - Junior Cloud Native Developer - Endava Vietnam**
* **Topic:** Kubernetes
* **Summary:** Discussed challenges in container management, introduced Kubernetes architecture and core components. Guided on using objects, manifests, kubectl, and demonstrated real-world scenarios, learning paths, and tools like Helm, K9S.

**2. Thinh Nguyen - DevOps Engineer**
* **Topic:** Infrastructure as Code (IaC) with Terraform on AWS
* **Summary:** Explained the concept and importance of IaC in modern infrastructure management. Compared popular tools and showcased a practical deployment on AWS using Terraform.

**3. Nguyen Ta Minh Triet - SAP Developer Intern, Bosch GSV**
* **Topic:** Introduction to Elixir
* **Summary:** Presented Elixir fundamentals, parallel programming, OTP for fault tolerance, and how Elixir is applied in DevOps pipelines, with live demos.
