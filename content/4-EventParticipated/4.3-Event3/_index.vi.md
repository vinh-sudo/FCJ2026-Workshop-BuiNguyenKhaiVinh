# “Cloud Mastery 2026 #3 AWS Security & Networking”

### Mục Tiêu Sự Kiện

- Trình bày tổng quan về IAM, mạng và bảo mật trên AWS.
- Trang bị kiến thức nền tảng về kiểm soát truy cập, thiết kế mạng và bảo vệ ứng dụng.
- Kết nối lý thuyết với thực tiễn qua các ví dụ và demo cụ thể.

### Thông Tin Diễn Giả & Chủ Đề

**1. Huynh Hoang Long & Dang Thi Minh Thu - FCAJ Cloud Engineer Ambassador**
* **Chủ đề:** IAM trên AWS
* **Nội dung:** Giải thích khái niệm IAM, cách quản lý danh tính và quyền truy cập trên AWS. Trình bày các best practices, cơ chế SSO, ứng dụng SCP và Permission Boundaries để kiểm soát quyền, cùng các kỹ thuật bảo mật như credential rotation và Access Analyzer. Có demo thực tế minh họa.

**2. Lam An Thinh & Nguyen Phan Quoc Viet - Security Engineer Intern**
* **Chủ đề:** Mạng trên AWS
* **Nội dung:** Tổng quan về networking trong AWS, hướng dẫn thiết kế VPC, phân bổ CIDR, chia Subnets, cấu hình IGW và Route Table, sử dụng VPC Endpoint để tối ưu hóa kết nối nội bộ.

**3. Lam Tuan Kiet - DevOps**
* **Chủ đề:** Bảo vệ mạng & ứng dụng trên AWS
* **Nội dung:** Giới thiệu các dịch vụ bảo mật như AWS WAF, AWS Shield, AWS Network Firewall, AWS Firewall Manager, tập trung vào bảo vệ hệ thống và ứng dụng trước các mối nguy.

### Nội Dung Nổi Bật

#### 1. IAM và kiểm soát truy cập
- Hiểu rõ vai trò của IAM trong AWS.
- Áp dụng best practices để quản lý quyền truy cập an toàn.
- Tìm hiểu SSO trong doanh nghiệp.
- Sử dụng SCP và Permission Boundaries để kiểm soát quyền nâng cao.
- Thực hiện credential rotation và Access Analyzer để tăng bảo mật.
- Xem demo thực tế.

#### 2. Mạng trên AWS
- Nắm vững cách hoạt động của networking trong AWS.
- Thiết kế VPC và phân bổ CIDR hợp lý.
- Chia Subnets cho hệ thống.
- Cấu hình IGW và Route Table để điều hướng traffic.
- Tối ưu kết nối nội bộ với VPC Endpoint.

#### 3. Bảo vệ mạng và ứng dụng
- Tìm hiểu cách sử dụng AWS WAF để bảo vệ web app.
- Hiểu vai trò của AWS Shield trong phòng chống DDoS.
- Ứng dụng AWS Network Firewall để kiểm soát lưu lượng mạng.
