# “Cloud Mastery 2026 #3 AWS Security & Networking”

### Event Goals

- Present an overview of IAM, networking, and security on AWS.
- Provide foundational knowledge on access control, network design, and application protection.
- Connect theory with practice through real-world examples and live demos.

### Speakers & Topics

**1. Huynh Hoang Long & Dang Thi Minh Thu - FCAJ Cloud Engineer Ambassador**
* **Topic:** IAM on AWS
* **Details:** Explained IAM concepts, identity and access management on AWS. Shared best practices, SSO mechanisms, use of SCP and Permission Boundaries for advanced access control, and security techniques like credential rotation and Access Analyzer. Included practical demonstrations.

**2. Lam An Thinh & Nguyen Phan Quoc Viet - Security Engineer Intern**
* **Topic:** Networking on AWS
* **Details:** Provided an overview of AWS networking, guided on VPC design, CIDR allocation, subnetting, configuring IGW and Route Tables, and using VPC Endpoints to optimize internal connectivity.

**3. Lam Tuan Kiet - DevOps**
* **Topic:** Securing Networks & Applications on AWS
* **Details:** Introduced security services such as AWS WAF, AWS Shield, AWS Network Firewall, and AWS Firewall Manager, focusing on protecting systems and applications from threats.

### Key Highlights

#### 1. IAM and Access Control
- Understand the role of IAM in AWS.
- Apply best practices for secure access management.
- Learn about SSO in enterprise environments.
- Use SCP and Permission Boundaries for advanced permissions.
- Implement credential rotation and Access Analyzer for enhanced security.
- Watch practical demos.

#### 2. Networking on AWS
- Grasp how networking works in AWS.
- Design VPCs and allocate CIDR blocks effectively.
- Subnet systems appropriately.
- Configure IGW and Route Tables for traffic management.
- Optimize internal connections with VPC Endpoints.

#### 3. Securing Networks and Applications
- Learn to use AWS WAF for web application protection.
- Understand AWS Shield's role in DDoS mitigation.
- Apply AWS Network Firewall to control network traffic.
